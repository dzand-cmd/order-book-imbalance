**Author:** dzand-cmd  
**Project Type:** Quantitative Research / Algorithmic Trading  
**Language:** Python  
**Status:** In Progress  

---

## Overview

This project explores **order book imbalance signals** for short-horizon predictive trading. The goal is to extract **alpha signals** from limit order book data to identify potential price movements over very short timescales (milliseconds to seconds).

It is designed as a portfolio project to demonstrate skills in:  
- Quantitative research  
- Python data analysis and visualization  
- Feature engineering on high-frequency data  
- Backtesting trading signals
